"Silly Oddities" - a mod that adds goofy compat with other silly mods & partial backport from vanilla.

![](https://fluffy-village.dev/assets/creations/silly_oddities/logo.png)

Using **[Polymorph](https://modrinth.com/mod/polymorph)** is recommended to avoid conflicts with crafts.

# Backports:

Backports were created from scratch using most of vanilla assets at the desire of developer and they differs from vanilla.

![](https://fluffy-village.dev/assets/creations/silly_oddities/backports.png)

Tuff blocks (from 1.21)  
Copper blocks (from 1.21)  
Plants (from 1.21.5)  
Bundles and their new variations  
Horse armor (from 1.21.11)  
New paintings from modern versions  
New crafts from modern versions  
New textures from modern versions  
Bat model (from 1.20.3)  
Empty air overlay (from 1.21.2)  

# Compatibility:

Compatibility with other mods are NOT mandatory, if you didn't install compatible mod item's wont be added to the game. 

![](https://fluffy-village.dev/assets/creations/silly_oddities/compatibility.png)

**[Wizard's Reborn](https://modrinth.com/mod/wizards-reborn)**, **[Malum](https://modrinth.com/mod/malum)**  
Wizard's Reborn wooden blocks from vanilla wood  
Wizard's Reborn wooden blocks from Malum wood
